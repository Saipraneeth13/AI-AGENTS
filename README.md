# AI Agent for Resume

A small project that provides an AI-powered assistant to help generate, review, and improve resumes.

## Features
- Generate resume content using prompts
- Review and suggest improvements
- Basic local CLI for testing prompts

## Quick Start (macOS)
1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd AI-agent_for_resume
   ```
2. Create a virtual environment and install dependencies:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
3. Run the main script or tests:
   ```bash
   python main.py
   pytest
   ```

## Usage
- Edit prompts and configuration in the `config/` or `src/` folder.
- Run `python main.py` to start the CLI flow.
